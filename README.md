# pandoc-skill

Convert documents between formats using pandoc.

## Download

You can get this skill in either of these ways:

### GitHub

Clone or download the repository from GitHub:

```bash
git clone https://github.com/hrkltz/pandoc-skill.git
```

If you prefer not to clone, download the repository as a ZIP from GitHub and extract it into your
skills directory.

### clawhub.ai

Open the skill page on [ClawHub](https://clawhub.ai/oliver-hrkltz/pandoc), download or install the skill from there, and add it to your
OpenClaw skills directory according to your local setup.

## Notes

This skill requires `pandoc` to be installed. PDF output also requires a supported PDF engine such
as `xelatex`, `lualatex`, `tectonic`, `pdflatex`, `weasyprint`, `wkhtmltopdf`, or `prince`.
